How to Process, Analyze and Visualize Data by Adam Marcus, Eugene Wu

Data science! Big data! Statistics! Infographics! Buzzword!

We will give you an introductory crash course to taking raw data files (e.g., text documents, scraped websites), extracting meaningful information from them, and visualizing the results. We'll cover tools to clean and convert raw data files into structured data, analyze the data using common database and statistical tools, and finally visualize the results with popular libraries.

If there's time, we'll show you how to crank through particularly large or dirty datasets using frameworks like MapReduce and tools like Amazon's Mechanical Turk.

This course is heavily hands-on. Each class will present a short demo or introduction of a new data processing technique, and spend the rest of the time applying it to actual data.
